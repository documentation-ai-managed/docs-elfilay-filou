## Upgrading to v1.2.0

Follow these steps to update Elfilay Filou and access the latest features.

<Steps>
  <Step title="Backup Your Data" icon="database">
    Export your current dashboards from `https://dashboard.example.com/export`.

    ```bash
    curl -H "Authorization: Bearer YOUR_TOKEN" \
         https://api.example.com/export > backup.json
    ```
  </Step>
  <Step title="Install Latest Version" icon="download">
    Update via your package manager.

    <CodeGroup tabs="npm,yarn,pnpm">
      ````bash
      npm install elfilay-filou@1.2.0
      ````

      ````bash
      yarn add elfilay-filou@1.2.0
      ````

      ````bash
      pnpm add elfilay-filou@1.2.0
      ````
    </CodeGroup>
  </Step>
  <Step title="Run Migration" icon="settings">
    Execute the built-in migrator.

    ```bash
    npx elfilay-filou migrate
    ```
  </Step>
  <Step title="Restart and Verify" icon="check-circle">
    Restart your application and test at `https://dashboard.example.com`.
  </Step>
</Steps>

<Callout kind="tip">
  Review deprecation notices in each release. Set `debug={true}` in your config for detailed logs during upgrade.
</Callout>
