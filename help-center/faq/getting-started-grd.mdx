## Quick Start

To use Elfilay Filou grd:

1. Sign up at `https://elfilayfilou.com/signup`
2. Log in and create your first **grd project**
3. Download the grd client from the dashboard

## First Steps

```bash
grdfilou login --token YOUR_API_TOKEN
grdfilou init myproject
```
Your grd instance is now ready to use.
