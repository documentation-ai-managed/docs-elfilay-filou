## Overview

Elfilay Filou empowers you to manage and visualize grid-based data effortlessly. As a SaaS platform, it provides intuitive tools for importing, editing, and analyzing tabular datasets in real-time. You handle large grids with low latency, apply filters, and generate insights without complex setups.

Whether you work with spreadsheets, databases, or custom data exports, Elfilay Filou streamlines your workflow. Access it via `https://dashboard.example.com` and start transforming raw data into actionable visuals today.

## Get Started Quickly

Follow these steps to create your first grid.

<Steps>
  <Step title="Sign Up" icon="user-plus">
    Visit `https://dashboard.example.com/signup` and create an account with your email.
  </Step>
  <Step title="Create a Grid" icon="plus">
    Click "New Grid" and upload a CSV file or enter data manually.
  </Step>
  <Step title="Visualize Data" icon="eye">
    Select columns and choose a chart type. Preview and share instantly.
  </Step>
</Steps>

Here is a simple example using the JavaScript SDK to fetch a grid:

<CodeGroup tabs="JavaScript,Python">
  ```javascript
  import { ElfilayFilou } from '@elfilay/filou-sdk';

  const client = new ElfilayFilou({ apiKey: 'YOUR_API_KEY' });
  const grid = await client.grids.get('grid-123');
  console.log(grid.data);
  ```
  ```python
  from elfilay_filou import ElfilayFilou

  client = ElfilayFilou(api_key="YOUR_API_KEY")
  grid = client.grids.get("grid-123")
  print(grid.data)
  ```
</CodeGroup>
